# UADE - Ingeniería de Datos 2
## Proyecto: Gestión de Proyectos (Front-End)
### Tecnologías: React, Vite, Tailwind CSS, Ant-Design, React Hook Form

---

## Instalación de dependencias:

```bash
npm install

```

## Iniciar server de desarrollo

```bash
npm run dev
```

## Compilación

```bash
npm run build
```
